How it works

Upload a raw CBCT scan (.nii). The model slices the volume, runs each slice through a trained U-Net, stitches the predictions back into 3D, then cleans up noise before rendering an interactive segmentation you can rotate and inspect.

  1. 1Upload a raw .nii CBCT volume
  2. 2U-Net runs inference slice-by-slice on GPU
  3. 3Noise removal + hole-filling cleans the mask
  4. 4Interactive 3D result + downloadable report
